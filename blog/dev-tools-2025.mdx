## 编辑器

目前主力是 **VS Code** + Vim 插件，偶尔用 Neovim 处理简单的文件编辑。

常用插件：

- GitHub Copilot — AI 补全
- Error Lens — 行内错误提示
- Pretty TypeScript Errors — 更可读的 TS 错误

## 终端

- **Warp** — 主力终端，AI 集成很方便
- **zsh** + oh-my-zsh
- **starship** — 自定义 prompt

## 常用 CLI 工具

| 工具 | 用途 | 替代 |
|------|------|------|
| `eza` | 列出文件 | `ls` |
| `bat` | 查看文件 | `cat` |
| `fd` | 搜索文件 | `find` |
| `rg` | 搜索内容 | `grep` |
| `fzf` | 模糊搜索 | — |
| `zoxide` | 快速跳转 | `cd` |

## 浏览器

Chrome + 这些扩展：

- React DevTools
- JSON Formatter
- Refined GitHub

## 笔记

- Obsidian 做知识管理
- Notion 做项目管理

工具是为了提升效率，找到适合自己的就好，不必追求最新最热。
