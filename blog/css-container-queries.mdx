## 什么是 Container Queries

Media Queries 基于视口宽度响应，而 Container Queries 基于父容器宽度响应。这对组件化开发来说意义重大。

## 基本用法

```css
.card-container {
  container-type: inline-size;
  container-name: card;
}

@container card (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 1fr 2fr;
  }
}

@container card (max-width: 399px) {
  .card {
    display: flex;
    flex-direction: column;
  }
}
```

## 与 Tailwind CSS 配合

Tailwind CSS v4 对 Container Queries 有很好的支持：

```html
<div class="@container">
  <div class="@sm:grid @sm:grid-cols-2">
    <!-- 根据容器宽度响应 -->
  </div>
</div>
```

## 实际场景

- 侧边栏收起/展开时，卡片自动调整布局
- 同一个组件在不同位置（主区域 vs 侧边栏）显示不同样式
- Dashboard 中的可拖拽面板

Container Queries 让真正的组件级响应式成为可能。
