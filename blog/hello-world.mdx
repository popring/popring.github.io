## 新的开始

经过一段时间的折腾，新博客终于上线了。这次选择了 Next.js 作为框架，告别了之前用了好几年的 Hexo。

### 为什么换

主要原因有几个：

1. 想要更灵活的自定义能力
2. Next.js 的 App Router 体验很好
3. 可以方便地加入动态功能

### 技术栈

- **框架**: Next.js 16
- **样式**: Tailwind CSS v4
- **部署**: Vercel
- **内容**: MDX

```tsx
// 一段简单的示例代码
export default function Page() {
  return <h1>Hello, World!</h1>
}
```
