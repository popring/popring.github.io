visibility: hidden、display: none、opacity: 0 区别

发表于 2020-10-16 更新于 2021-03-28 分类于 css

结论

做了以下的实验，可以得出结论，

  • visibility: hidden; 只是在视觉上隐藏，但没有影响布局，但绑定的事件也无法触发。
  • display:none; 元素被彻底隐藏，在页面上完全不显示，不占位，无宽高
  • opacity: 0; 只会修改元素透明度，并不影响布局及事件触发。
